

# Strava Club Battlegrounds – Product Pitch Website

A modern, high-conversion single-page portfolio website with Strava-inspired branding (primary orange #FC4C02, white backgrounds, dark charcoal text, bold sans-serif typography).

## Sections to Build

1. **Hero** – Centered headline "Strava Club Battlegrounds: The Sponsored Series" over a dark gradient background, sub-headline, body copy, and a CTA button linking to the prototype section. Smooth entrance animations.

2. **The Context & The Gap** – Two-column layout: left text with bullet points about the Gen Z run club observation and the leaderboard gap; right side with a placeholder image of a diverse run club.

3. **The Solution** – Centered headline with a bold, high-contrast feature card describing Sponsored Club Battlegrounds and the B2B model.

4. **Product Strategy (DVF Framework)** – Three-column grid with icons for Desirability, Viability, and Feasibility, each with the provided copy.

5. **Interactive Prototype & User Flow** – Full-width section with the 5-step user flow (Discovery → Opt-In → Execution → Validation → Conversion) and a large Figma embed placeholder below.

6. **Execution (Metrics & GTM)** – Two-column layout: left side lists key metrics, right side shows a phased GTM timeline (Beta → Evangelism → Global rollout).

7. **About Me** – Left-aligned profile photo placeholder, right-aligned bio text, and buttons for LinkedIn and Resume download.

## Design Details
- Strava orange (#FC4C02) as primary color, updated in the Tailwind theme
- Smooth scroll navigation with a sticky top nav bar linking to each section
- Subtle fade-in/slide-up animations on scroll for each section
- Clean card-based layouts with generous whitespace
- Responsive design for mobile and desktop
- Placeholder images with relevant alt text (user can swap in real images later)
